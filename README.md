# Loving Homes - Dog Hotel Website (School Project)

موقع ويب مكوّن من 8 صفحات باستخدام HTML/CSS/JS.
- واجهة عربية (RTL) + نسخة إنجليزية (LTR) داخل مجلد `en/`.
- زر تبديل الوضع (نهاري/ليلي) مع حفظ الإعداد في المتصفح.
- زر تبديل اللغة (AR/EN) مع حفظ الإعداد في المتصفح.

## الصفحات (Arabic)
index.html, about.html, services.html, packages.html, gallery.html, reviews.html, faq.html, contact.html

## الصفحات (English)
داخل `en/`:
index.html, about.html, services.html, packages.html, gallery.html, reviews.html, faq.html, contact.html

## التشغيل
افتح `index.html` مباشرة أو شغّل خادم محلي داخل المجلد:

```bash
python -m http.server
```

ثم افتح المتصفح على:
`http://localhost:8000/`
